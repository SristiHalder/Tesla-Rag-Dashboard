# Tesla Intelligence Agent — RAG Dashboard

This dashboard is an interactive Retrieval-Augmented Generation (RAG) application built using **Streamlit**, **OpenAI**, and **TF-IDF retrieval**.  
It transforms Tesla’s corporate filings into a transparent, verifiable question-answering system.

Users can explore:
- Tesla’s business segments  
- Financial performance  
- ESG priorities  
- Risk factors  
- Operational insights  

The system retrieves relevant evidence from curated text chunks and generates grounded answers using OpenAI’s API, citing chunk IDs for transparency.

---

## Features

### **1. Retrieval-Augmented Generation (RAG)**
- TF-IDF similarity search  
- Evidence-based responses  
- Transparent citations  

### **2. Tesla KPI Dashboard**
- Revenue  
- Margins  
- Segment performance table  
- Automatically loaded from structured KPIs  

### **3. Interactive Q&A Agent**
Users can:
- Select predefined questions  
- Ask custom questions  
- View retrieved evidence  
- See a grounded final answer  

### **4. Streamlit Web App**
- Clean minimal dark theme  
- Fully deployable on Streamlit Cloud
